# Xiaohongshu AI Agent

> An open-source AI agent for Xiaohongshu content creation and marketing automation.

小红书 AI 内容运营助手，帮助创作者和中小商家使用 AI 提升内容生产效率。


## 🌟 项目介绍

Xiaohongshu AI Agent 是一个开源 AI Agent 项目。

目标是帮助：

- 小红书创作者
- 电商卖家
- 内容运营人员

通过 AI 自动完成：

- 商品卖点分析
- 爆款标题生成
- 种草笔记创作
- 短视频脚本生成
- 评论互动优化


## 🚀 Features

计划支持：

- [ ] AI 标题生成
- [ ] AI 笔记生成
- [ ] 商品分析 Agent
- [ ] 图片内容理解
- [ ] 爆款内容分析
- [ ] 多 Agent 协作
