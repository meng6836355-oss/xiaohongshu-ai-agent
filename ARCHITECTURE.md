# Architecture

Xiaohongshu AI Agent is designed as a modular AI agent system for content creation and marketing automation.

## Core Components


## 1. Content Generation Agent

Responsible for:

- Xiaohongshu title generation
- Note writing
- Content rewriting
- Marketing copy creation


## 2. Product Analysis Agent

Responsible for:

- Product feature extraction
- Selling point analysis
- Target audience analysis


## 3. Prompt System

Manages:

- Writing templates
- Content strategies
- Brand tone


## 4. Future Multi-Agent System

Planned agents:

- Content Agent
- Data Analysis Agent
- Marketing Strategy Agent


## Development Direction

The project aims to build an open-source AI marketing assistant that helps creators and small businesses automate content workflows.
