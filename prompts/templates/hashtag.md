# Xiaohongshu Hashtag Template


## Purpose

Generate relevant hashtags to improve content discovery.


---

# Input

```json
{
  "topic": "",
  "industry": "",
  "audience": ""
}
```


---

# Output

```json
{
  "primary_tags": [],

  "long_tail_tags": [],

  "community_tags": []
}
```


---

# Tag Categories


## Primary Tags

Broad topic tags.

Example:

```
AI
人工智能
效率工具
```


---

## Long Tail Tags

Specific search intent.

Example:

```
AI办公技巧
ChatGPT使用教程
程序员效率提升
```


---

## Community Tags

Interest-based tags.

Example:

```
自媒体成长
创业日常
职场提升
```


---

# Rules

Avoid:

- Irrelevant trending tags
- Excessive tags
- Spam keywords


Recommended:

5-10 highly relevant tags.
